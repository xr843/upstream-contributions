## 自动化 / Automation

`scripts/update_contributions.py` 通过 GitHub Search API 拉取 [@xr843](https://github.com/xr843) 在所有外部仓库的 PR，按规则排序后回写本 README。

```bash
# 本地手动跑（需 `gh auth login` 或 `GITHUB_TOKEN` 环境变量）
python scripts/update_contributions.py
```

GitHub Actions 每周一 09:00 (Asia/Shanghai) 自动重跑，见 `.github/workflows/update.yml`。

## 排序逻辑

- **stars × PR count**：避免"低星项目刷量"上头条；也避免"只算 stars"埋没深耕的中小项目
- 每个项目内按 PR 号倒序
